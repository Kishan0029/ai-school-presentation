# AI School Presentation

A premium, interactive AI presentation built with pure HTML, CSS, and JS. 

## How to use:
1. Double-click the **`index.html`** file to open it in any modern browser (Chrome, Edge, Safari).
2. Press **`F`** to go Fullscreen.
3. Use **Right Arrow** or **Spacebar** to move to the next slide.
4. Use **Left Arrow** to go back.

No servers, no installations, no frameworks required. Everything runs locally in the browser.
